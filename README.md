#  SQL Monthly Revenue Analysis

##  Objective
Analyze monthly revenue and order volume from online sales data.

##  Tools Used
-  MySQL
- GROUP BY, EXTRACT, ORDER BY, LIMIT

##  Dataset
online_sales_dataset.csv

##  Analysis Performed
- Monthly revenue calculation using SUM()
- Order volume using COUNT(DISTINCT order_id)
- Top 5 months by revenue
- Year-wise grouping

##  Key Insights
- Identified top-performing months
- Observed revenue trends across the year
- Compared revenue with order volume
